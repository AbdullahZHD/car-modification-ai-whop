# Welcome to Whop Next.js App Template

Build powerful applications on the Whop platform with this comprehensive Next.js template. This template provides everything you need to get started with creating apps for the Whop ecosystem.

## Features

- **Next.js 15** with App Router and Turbopack support
- **Whop SDK** for seamless platform integration
- **TypeScript** for type safety and better developer experience
- **Tailwind CSS** for modern, responsive styling
- **Prisma ORM** for database management
- **Biome** for code formatting and linting
- **OpenAI Integration** for AI-powered features
